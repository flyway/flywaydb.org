First Steps

This guide will help you get started with Flyway. We'll go through the steps to get Flyway up and running and having it execute a first migration.

Start by choosing how you wish to run Flyway:

Client Description
API Integrate Flyway directly into your application
Maven Run Flyway as a Maven plugin
Command-line Run Flyway with its independent command-line tool
Ant Run Flyway through Ant tasks
