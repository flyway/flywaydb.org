Sponsor a Feature or Bug fix

If there is a feature or a fix your organisation wants that fits into the product vision, and it hasn't been implemented yet, you also have the option to pay a bounty to have it implemented sooner.

This has several advantages over implementing it yourself:

  • No need to dig in the framework internals
  • No need to set up the dev environment
  • No need to actually implement and debug it
  • No need to maintain your own branch (with the merge hassles for subsequent releases)
  • No need to wait until it gets approved into the main distribution

Instead it will be developed, integrated and documented in the next official release for you.

The price will cover developer time and will depend on the scope of the feature.
